# ビデオチュートリアル

このページでは、Coasts の YouTube チャンネルにある公式 Coasts チュートリアル動画をまとめています。

ドキュメントの残りを読む前に手早く動画でのウォークスルーを見たい場合は、まず概要と入門動画から始め、その後に以下の機能別トピックへ進んでください。

## リンク

- [Coasts YouTube channel](https://www.youtube.com/@coasts-dev) - Coasts 動画の公式チャンネル。
- [Coasts playlist](https://www.youtube.com/playlist?list=PLplJaH2mu9vQFqnyJHrrls6JU0yt9CUuw) - チュートリアルの全プレイリストを1か所にまとめたもの。
