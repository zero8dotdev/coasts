# Видеоуроки

На этой странице собраны официальные обучающие видео Coasts с YouTube-канала Coasts.

Если вы предпочитаете короткий видеообзор перед тем, как читать остальную документацию, начните с видео-обзора и видео по началу работы, затем переходите к тематическим роликам по конкретным функциям ниже.

## Ссылки

- [Coasts YouTube channel](https://www.youtube.com/@coasts-dev) - Официальный канал с видео Coasts.
