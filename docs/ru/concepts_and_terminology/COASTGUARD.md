# Coastguard

Coastguard — это локальный веб-интерфейс Coast (представьте: интерфейс Coast в стиле Docker Desktop), работающий на порту `31415`. Он запускается из CLI:

```bash
coast ui
```

## Для чего полезен Coastguard

Coastguard предоставляет вам визуальную поверхность управления и наблюдаемости для вашего проекта:

- Просматривайте проекты, экземпляры, статусы, ветки и состояние checkout.
- Изучайте [сопоставления портов](PORTS.md) и переходите напрямую к сервисам.
- Просматривайте [логи](LOGS.md), статистику выполнения и инспектируйте данные.
- Просматривайте [сборки](BUILDS.md), артефакты образов, метаданные [томов](VOLUMES.md) и [секретов](SECRETS.md).
- Открывайте документацию внутри приложения во время работы.

## Связь с CLI и демоном

Coastguard не заменяет CLI. Он дополняет его как интерфейс для человека.

- [`coast` CLI](CLI.md) — интерфейс автоматизации для скриптов, агентных рабочих процессов и интеграций с инструментами.
- Coastguard — интерфейс для человека для визуальной инспекции, интерактивной отладки и повседневной операционной видимости.
- Оба являются клиентами [`coastd`](DAEMON.md), поэтому они остаются синхронизированными.
